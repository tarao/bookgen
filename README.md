bookgen
=======

Preview
-------

### Requirements

- `make`
- `bash`
- `docker`
- `go`
- `python3`

### Command

```
make preview
```

- Preview in your browser
- With auto reload
- Ctrl+C to stop preview

Rendering
---------

### Requirements

- `make`
- `bash`
- `docker`

### Preprocessed markdown (plain text)

```
make markdown
cat dist/book.md
```

### HTML

```
make html
cat dist/book.html
```
